Scanner Betano - pronto para Render.com

Instruções rápidas:
1) Faça upload deste repositório no GitHub.
2) Em Render, crie New -> Web Service -> Connect GitHub -> selecione este repo.
3) Escolha Environment = Docker, Branch = main.
4) Deploy. Aguarde build (pode demorar, instala Chromium).
5) Abra a URL pública e clique em 'Atualizar'.

Observações:
- O scraper usa Puppeteer; se a Betano mudar a estrutura, pode precisar ajustes.
- Este serviço não atualiza automaticamente; apenas quando clicar 'Atualizar'.
